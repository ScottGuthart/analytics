File labeling instructions

Note: It's okay if your ranking and MaxDiff data are in the same file.

MaxDiff Design

Make sure the first sheet of the excel file contains the design

MaxDiff File

Best ranks must start with:*MDBEST

Worst ranks must start with: *MDWORST

Responseid: responseid

Respid: respid

MaxDiff Version: 'hVersion'

Notes: The script expects both a respid and response id column. For the MDBEST and MDWORST columns, you don't necessarily need to include the task number after *MDWORST -- the script assumes that they appear in the data in the same order as the design. So that the best and worst items for the first task appear before the best and worst items for the second task.

Ranking File

Best ranks: *RANKBEST_[item#]

Worst ranks: *RANKWORST_[item#]

Worst ranks: *RANKWORST_[item#]

Notes: *RANKBEST_1* will correspondent to the ranking of item 1 not the item a respondent ranked as 1. This should be the same format you receive the data in so you don't need to do any additional setup here.

Win Loss Rates File

All you need for this file is a column with the header Wins where the first row is the number of times you'd like the best of the best item to win and the last row with the number of times you'd like the worst of the best items to win.

Note: When generating pairs for the worst items, the script reverses the order
